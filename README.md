# CSCI291-Project
Zainab Khan 8442861 || Neha Shababudeen 8388635

The goal of this project is to design and program an autonomous maze-solving robot 
using Webots. The focus is to create an efficient robot capable of navigating complex 
mazes and record the varying light intensities throughout to identify the point with the 
maximum light intensity. 

This is achieved by integrating several proximity and light sensors, and a GPS, along 
with the use of the right-hand wall following algorithm. 
The implementation is carried out in C, and not hardcoded for any specific maze 
configuration.
